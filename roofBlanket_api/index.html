(User) Get all User Information

Description
Returns a collection of all user information in JSON format.

HTTP request
Mode Method API Endpoint URL
Local GET Assume roofblanket folder is in the same directory as your HTML/JS files:
rooofblanket_api/api/functions/getusersinfo.php

Assume roofblanket folder is directly under the webroot of WAMP/MAMP and your HTTP port is 80:
http://localhost/roofblanket_api/api/functions/getusersinfo.php
